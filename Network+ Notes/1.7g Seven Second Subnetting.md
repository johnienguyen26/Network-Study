## The seven second subnetting process
1. Convert IP address and subnet mask to decimal
  - Use chart to convert between CIDR-block notation and decimal
  - Same chart also shows the number of devices per subnet
2. Determine network/subnet address
  - Second chart shows the starting subnet boundary
3. Determine broadcast address
  - Second chart shows the ending subnet boundary
4. Calculate first and last usable address
  - Add one from network address, subtract one from broadcast address
## Seven second subnetting:(examples)
### RULES TO REMEMBER:
1. Covert address and mask to decimal
2. To calculate the network address: If mask is 255, bring down the address, If mask is 0, use the 0
3. To calculate the broadcast address: If mask is 255, bring down the address, If mask is 0, use 255
4. First IP is NETWORK address +1
5. Last IP is BROADCAST address -1
